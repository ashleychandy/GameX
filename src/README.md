# GameX Frontend

## Setup
1. Clone repository
2. Copy .env.example to .env and fill in values
3. Install dependencies: `npm install`
4. Run development server: `npm run dev`

## Architecture
- React 18 with Hooks
- Ethers.js for Web3
- Styled Components for styling
- React Router for routing
- Context API for state management

## Testing
Run tests: `npm test`
